Cairo's Data Change Techniques

Data Sonification

This means that you use sounds to communicate what the data means. So, you would play different notes/tones to represent different changes being represented in the data visualization.

Proportional Representation

This means using different areas to represent the data. It will allow the reader to compare the whole to its smaller parts.

Connected Scatter Plot

This is a graph that looks like a scatter plot, except that all of the dots are connected from the first change to the last. This shows changes through comparing where the scatter plot started off at to where it is now.
